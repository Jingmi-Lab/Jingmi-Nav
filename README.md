## 提示
开始使用Vue+VueCli3+VueRouter+MongoDB**重写**导航站，更新日志在新的分支下，到时候完工再合并。

你如果有兴趣可点击star或者watch长期关注，如果你有问题可在issues下留言，谢谢！

## 功能特性
- Vue单页面应用
- 兼容各主流浏览器和移动端(IE死开)
- 用户可提交网站，后台有页面审核用户提交的网站
- 提交留言
- 使用MongoDB存储网站数据
